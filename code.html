  • Code For Good

    09-10 December 2016 - London

    With 5 other friends, we took part in this intensive coding challenge. The purpose was to create a full-stack application for the charity Edulution, and we had only one night to finish it. They wanted to develop a program to help children in Zambia to learn. Mathematics, lietracy, agriculture, our application had to cover a lot of subject possibilities. And we had to adapt for children who were not able to read. This challenge was full of positibe energy!
